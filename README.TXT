Bienvenue dans le monde de One Piece, le célébre manga de piraterie !
Vous incarnez Luffy et son équipage, un groupe de pirate
Vous recherchez les ponéglyphes,ces indicateurs de l'emplacement du trésor ultime : le One Piece
Vous êtes sur le territoire de Big Mom, un des 4 plus puissants pirates du monde à la recherche de l'un d'entre eux
Vous êtes séparés du groupe, tous présent sur le bateau, prêt à partir au sud de l'ile Confiture
Votre but sera donc de ramener le ponéglyphe sur le bateau
Mais attention : Big Mom a installé dans certains coins des tours de surveillance, appelées Tarta
Si vous passez 3 fois devant l'un d'entre eux, Big Mom vous trouvera et vous tuera
Bonne chance et 幸運 (bon courage)
